Project 3: Intel Sustainability Journey

An interactive, responsive webpage that showcases Intel’s sustainability goals in a dynamic timeline format.

This project focuses on combining structured content, smooth UI interactions, and responsive design principles to create an engaging user experience across both desktop and mobile devices.

Objective

Design and build a timeline-based webpage that:

Highlights Intel’s sustainability milestones and commitments

Uses hover effects and transitions to enhance interactivity

Adapts seamlessly to different screen sizes

Maintains a clean and accessible layout

Key Features

Interactive timeline layout

Responsive design for desktop and mobile

Hover effects and smooth transitions

Structured, accessible content presentation

Technologies Used

HTML

CSS

Responsive design techniques (media queries, flexible layouts)

Optional AI-assisted styling and layout refinement
